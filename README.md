# OptiCore Website

This repository contains the source code for the **OptiCore** marketing website. The static site showcases the company's services, skills, and projects with a clean responsive design and a dark mode toggle.

## Project Structure

- `index.html` – Main landing page with introduction, skills, portfolio, and contact sections.
- `OurProjects.html` – Dedicated page listing additional projects with live links and GitHub repositories.
- `style.css` – Styling file shared across pages, includes variables, layout rules, responsive media queries, and dark mode overrides.
- `img/` – Directory containing logo and project screenshot images used in the site.

## Features

- Responsive layout using CSS Grid and flexbox for mobile and desktop devices.
- Dark mode toggle (stores preference in `localStorage`).
- Navigation header with smooth scrolling to page sections.
- Portfolio cards linking to live demos and GitHub repositories.
- Contact form layout (no backend integration included).
- Consistent typography using Google Fonts and icons via Font Awesome.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern browser to view the homepage.
3. Use `OurProjects.html` to explore additional project cards.
4. Edit HTML or CSS files to customize content, styles, or add new projects.

## Customization & Development

- Update the nav links or sections in the HTML files as needed.
- Add new project cards by following the existing markup structure in the portfolio grid.
- Modify styling variables in `style.css` to adjust colors, fonts, or spacing.

## Deployment

Since this is a static site, it can be hosted on any static file server or platforms like GitHub Pages, Netlify, or Vercel. Simply push the files to a repository and configure the hosting service to serve from the root.
